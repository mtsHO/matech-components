# Typography

O `Typography` centraliza a exibicao de titulos, subtitulos e paragrafos da `@mateusho/matech-components`.

Ele foi pensado para dar consistencia sem depender de uma biblioteca de UI externa.

## Importacao

```tsx
import { Typography } from '@mateusho/matech-components';
```

## Uso rapido

```tsx
<Typography variant="h3">Feed da comunidade</Typography>
<Typography color="muted" variant="body1">
  Veja o que esta acontecendo na comunidade.
</Typography>
```

## Variantes

As variantes seguem uma linha familiar ao MUI:

- `h1` a `h6`
- `subtitle1`
- `subtitle2`
- `body1`
- `body2`
- `caption`
- `overline`

Sugestao extra da lib:

- `lead`: texto de introducao com mais destaque

## Personalizacao por tema

O `Typography` usa os tokens de `global palette` para cores e os tokens de `typography.variants` para tamanho, peso e espacamento.

### Exemplo com `createMatechTheme`

```tsx
import {
  MatechThemeProvider,
  Typography,
  createMatechTheme,
} from '@mateusho/matech-components';

const theme = createMatechTheme({
  global: {
    brand: '#31408B',
    brandStrong: '#283471',
    brandForeground: '#FFFFFF',
    surface: '#FFFFFF',
    surfaceMuted: '#F7F9FF',
    border: '#DEE4F5',
    borderStrong: '#C7D1EC',
    text: '#2A356E',
    textMuted: '#66719A',
  },
  typography: {
    variants: {
      h3: {
        fontSize: '1.75rem',
        fontWeight: 800,
        letterSpacing: '-0.02em',
      },
      body1: {
        lineHeight: 1.7,
      },
    },
  },
});
```
